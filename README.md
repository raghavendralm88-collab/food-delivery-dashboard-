## 📊 Project Overview

This project features an interactive Power BI dashboard built using Zomato food delivery data to analyze business performance and customer trends. The dashboard includes visualizations such as bar charts, pie charts, and KPI cards to highlight key metrics like total orders, popular food categories, customer segmentation, and restaurant performance.

It uses filters and slicers to allow dynamic exploration of data, helping users identify patterns and make informed decisions. The project demonstrates strong skills in data cleaning, data modeling, and visualization, along with the ability to convert raw data into meaningful business insights using Power BI.
